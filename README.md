<h3> Steps <h3>
<h4> 1. Clone this repository <h4>

```
 $ git clone https://github.com/mjulialobo/MyFlix
```

<h4> 2. Running the application on your terminal <h4>

```
#install the dependencies
$ npm install

#start the server
$ npm start

```
